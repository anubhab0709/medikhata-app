# 💊 MediKhata

A modern pharmacy ledger and customer management application built with React, Node.js, Express, and MongoDB.

## ✨ Features

- 🔐 Secure Email Authentication with OTP Verification
- 👥 Customer Management
- 💰 Due & Advance Tracking
- 📒 Customer Ledger
- 🔔 Payment Reminder Management
- 📊 Dashboard Analytics
- 📄 Invoice & Bill Management
- 🌙 Modern Responsive UI
- 📱 Mobile Friendly
- ⚡ Fast React + Vite Frontend
- ☁️ Cloud Ready (Render + MongoDB Atlas)

---

## 🚀 Getting Started

### Clone

```bash
git clone https://github.com/yourusername/medikhata.git
cd medikhata
```

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the backend folder.

Example:

```env
PORT=4000

MONGODB_URI=your_mongodb_uri

JWT_SECRET=your_secret

RESEND_API_KEY=your_resend_key

RESEND_FROM=MediKhata <noreply@yourdomain.com>

CLIENT_URL=http://localhost:5173
```
